# Movie Library — GUI-приложение для управления коллекцией фильмов

## Описание
Приложение на Python (Tkinter) для хранения и фильтрации фильмов по жанру и году выпуска. Данные сохраняются в формате JSON.

## Функционал
* Добавление фильмов (название, жанр, год, рейтинг);
* Фильтрация по жанру (выпадающий список);
* Фильтрация по году (точный год);
* Сохранение/загрузка данных в `movies.json`;
* Валидация ввода (год — число, рейтинг 0–10).

## Требования
* Python 3.8+;
* `json`, `os` — стандартные модули Python.

## Запуск
1. Клонируйте репозиторий: `git clone <ссылка на репозиторий>`.
2. Перейдите в папку проекта: `cd Movie-Library`.
3. Запустите приложение: `python main.py`.

## Структура проекта
* `main.py` — основной код GUI;
* `movies.json` — хранилище данных о фильмах;
* `.gitignore` — правила игнорирования файлов для Git;
* `README.md` — этот файл.

## Примеры использования
* Добавьте фильм: заполните форму и нажмите «Добавить фильм».
* Фильтруйте по жанру: выберите жанр из выпадающего списка и нажмите «Фильтровать по жанру».
* Фильтруйте по году: введите год и нажмите «Фильтровать по году».
